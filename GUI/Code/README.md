### Start project

1. Install dependencies with:
   ```bash
   npm install
   ```

2. Start the local development server with:
   ```bash
   npm run dev
   ```

Documentation, guides, examples and tutorials are available on [ui.vuestic.dev](https://ui.vuestic.dev)
